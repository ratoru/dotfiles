# Keymap

> **Generated by `deno task docs` ‚Äî do not edit by hand.** Edit `rules.ts` and
> re-run `deno task docs`. See `README.md` for setup and how the layers work.

**Right Command** is the action-layer key: hold it and press a key below. Tapping
Right Command alone sends Escape.

**Caps Lock** sends Escape when tapped and acts as Left Control when held. While a
sticky sublayer is active, Caps Lock cancels it instead.
